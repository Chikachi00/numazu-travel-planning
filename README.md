# 沼津店铺查询

静岡県沼津市の店舗・施設の営業日を日付で一括確認できる静的Webアプリです。

## 功能

- 选择日期后自动判断每家店当天是否开门
- 定休的店铺变灰显示，不隐藏
- 支持所有定休格式：单日、多日、第N周、月～金范围
- 按地区 / 类型筛选，支持关键词搜索
- 周末自动切换至周末营业时间

## 数据范围

117 家设施，覆盖沼津港、内浦、狮香、北口、南口、西北、あげつち商店街等主要区域。

## 更新店铺数据

所有数据在 `index.html` 底部的 `RAW` 数组中，每行格式如下：

```
[id, 店名, 地区, 类型, 开门时间, 关门时间, 周末时间, 定休日, 门票价格, 周末价格, 备注]
```

示例：

```js
[72, '魚河岸丸天', '沼津港', '餐馆', '10:00', '21:00', null, '木', null, null, '祝日は水曜休'],
```

### 定休日写法

| 写法 | 含义 |
|------|------|
| `'火'` | 每周二 |
| `'月・火'` | 每周一和周二 |
| `'木/日/祝'` | 周四、周日、节假日 |
| `'第3木'` | 每月第3个周四 |
| `'2・4土'` | 每月第2、4个周六 |
| `'月~金'` | 周一至周五 |
| `'不定休'` | 不定期休息（显示为黄色） |
| `'無'` | 无定休日 |
| `null` | 信息不明（显示为灰色待确认） |

## 技术说明

纯静态单文件，无框架，无依赖，无构建工具。

- 营业日判断：JavaScript `Date.getDay()`
- 第N曜日判断：`Math.ceil(date.getDate() / 7)`
- 字体：Google Fonts Noto Sans SC（需联网加载）
